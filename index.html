API documentation

Pair cart

use this URL to pair a cart with user

GET request;
http://mcprojectserver.appspot.com/paircart?cartid=<cart ID : alphanumeric>&userid=<registration ID : String>

response :
{
    "status" : "OK"
}

Get product information

use this URL to get location information about a product

GET request
http://mcprojectserver.appspot.com/getproductinformation?cartid=<cart ID : alphanumeric>&productname=<product name : String>

response :
{
    "shelf info": "3rd shelf", 
    "aisle": 4
}
